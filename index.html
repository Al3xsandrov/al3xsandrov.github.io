Video surveillance system

One of the side-projects that became commercial - a system for controlling surveillance cameras. I developed an integrated video surveillance system based on OpenCV and FFmpeg, providing asynchronous processing of video streams and performance optimisation through multi-threading. The result is an efficient solution for reliable monitoring and analysis of video data. Currently, it is the subject of team development and is being actively enhanced with new functionality.

Soundcloud music bot

I have created a Telegram bot that allows users to find and listen to music from SoundCloud. I have implemented AIOGram to interact with the Telegram API and integrated YouTube-DL, Requests and other tools for uploading and processing audio files. The result is a convenient and functional solution for music listening right in Telegram. This project was developed for personal use only, with no profit or commercial gain. The creation of the bot was my personal research project, which aimed to expand my programming skills and integrate various APIs.

EasyOCR App

While working on another project, I learned about text and image recognition technologies. This was the motivation for creating the application. EasyOCR App was created for educational purposes, implementing essential functionality without requiring GPU. It utilizes the EasyOCR library for text recognition on images and PyQt5 for GUI development. Compatible across various operating systems, it displays results within the application window and offers the option to save results in a .txt file.

Simple Parser App

The Simple Parser App was developed with educational objectives in mind. It enables users to search for jobs by parsing popular websites. It uses BeautifulSoup4 for web scraping, PyQt5 for GUI development, and Requests for HTTP requests, offering cross-platform compatibility. Results are conveniently saved to an .xlsx file, while the app boasts an intuitive interface with minimal settings for ease of use.

ParkingBot

During my Master's degree in Software Engineering, I was involved in the development of a parking control system. The ParkingBot was developed using OpenCV and NumPy as its foundation. The result is a software product that assists drivers in locating available parking spaces. The objective of my qualification work was to investigate the most effective detection methods for this system. Also the implementation of a telegram bot that allows users to interact with the system.

Numerical Assistant

The numerology calculator is an additional feature of a comprehensive astrological and numerological service delivery system. The result of this development is an application that is able to identify the querum based on the data entered by the user. In response to customer demand, the application is now available in multiple languages. In comparison to other similar systems, the calculation algorithm is significantly accelerated.

Electric Load Analysis System

By analysing the electrical load schedules, the engineer determines the future operating modes of the power systems. The objective of creating an electrical load analysis system was to reduce the likelihood of human error in complex mathematical calculations. By automating these processes, it was possible to reduce the workload of highly qualified engineers and increase the speed of decision-making. The project used SQLAlchemy for efficient interaction with databases. Since this project, I have begun to utilise Python as a tool for data processing and rapid prototyping, which has proven to be an invaluable asset in my professional work.
